Part 1 — 50분

ChatGPT 개인 맞춤 설정

ChatGPT는 새 대화를 열 때마다 사용자를 처음 만난 것처럼 인식합니다. 맞춤 설정은 이 문제를 해결하는 기능입니다.

Core Concept
맞춤 설정 = 모든 대화에 자동으로 적용되는 나만의 배경 정보와 응답 규칙
한번 설정하면 새 대화를 열 때마다 ChatGPT가 자동으로 참조합니다. 매번 "한국어로 답해줘", "나는 이런 사람이야"를 반복할 필요가 사라집니다.

출처: OpenAI 공식 블로그, "Custom Instructions for ChatGPT" (2023.07 공개, 이후 전체 사용자 확대)

설정 전과 후의 차이

같은 질문 "블로그 글 써줘"에 대한 응답 비교. 맞춤 설정 하나로 결과가 완전히 달라집니다.

설정 전

처음 가는 식당

메뉴판을 처음부터 봐야 하고, 취향을 매번 설명해야 합니다. 원하는 결과를 얻으려면 추가 질문이 반복됩니다.

설정 후

단골 식당

"오늘도 순두부 안 매운 걸로요?" — 나의 선호를 이미 알고 있어, 한마디면 원하는 결과가 나옵니다.

맞춤 설정의 구조

설정 화면에는 두 개의 입력란이 있습니다. 각각의 역할이 다릅니다.

실제 ChatGPT 맞춤 설정 화면의 구조를 재현한 모형입니다

입력란 1

나에 대한 정보

ChatGPT가 나를 이해하기 위해 필요한 맥락 정보입니다. 직업, 관심사, 언어, AI 사용 수준을 포함합니다.

입력란 2

원하는 응답 방식

ChatGPT의 답변 스타일을 제어합니다. 답변 길이, 말투, 형식, 금지 사항을 지정할 수 있습니다.

설정 경로

PC
프로필 아이콘
화면 우측 상단
→
ChatGPT 맞춤 설정
클릭하여 진입
앱
메뉴 > 프로필
좌측 상단 메뉴
→
개인 맞춤 설정
탭하여 진입

입력란 1 — 나에 대한 정보: 4가지 핵심 항목

항목 1
내가 하는 일
직업, 역할, 직책
항목 2
관심사 / 목적
현재 집중하는 일
항목 3
선호 언어
한국어 사용 명시
항목 4
나의 AI 수준
초보 / 중급 / 전문가
예시 A — 자영업자
1소규모 꽃집을 운영하는 자영업자
2꽃 배달 서비스 준비, SNS 마케팅
3한국어로 대화합니다
4AI는 처음 사용합니다
예시 B — 직장인
1중소기업 마케팅 팀장, 7년차
2보고서 작성, 경쟁사 분석
3한국어로 대화합니다
4AI를 가끔 사용해 봤습니다

입력란 2 — 원하는 응답 방식

답변 길이
간결하게 vs 상세하게
말투
격식체 vs 친근한 톤
형식
목록 / 표 / 문장형
금지 사항
이모지 금지, 영어 금지 등
입력란 2 작성 예시
항상 한국어로 답변해 주세요.
어려운 용어가 나오면 쉬운 말로 풀어서 설명해 주세요.
답변은 목록 형태로 정리해 주세요.
이모지는 사용하지 마세요.
주의: 맞춤 설정은 새 대화를 열어야 적용됩니다. 기존에 열려 있는 대화에는 반영되지 않습니다.
팁: 구체적으로 쓸수록 결과가 좋습니다. "직장인"보다 "중소기업 마케팅 팀장"이 훨씬 정확한 답변을 이끌어냅니다. 원하지 않는 것("영어 섞지 마세요")도 명시하면 효과적입니다.

Part 2 — 50분

나에게 맞는 GPTs 만들어보기

맞춤 설정이 ChatGPT의 '기본 성격'이라면, GPTs는 특정 업무 전용으로 만든 '별도의 AI 비서'입니다.

Core Concept
GPTs = 특정 목적에 맞게 역할, 규칙, 지식을 미리 설정해 둔 맞춤형 ChatGPT 챗봇
일반 ChatGPT에게는 매번 맥락을 설명해야 하지만, GPTs는 한 번 만들어두면 열자마자 전문적인 도움을 받을 수 있습니다. 파일(PDF, 엑셀 등)을 학습시킬 수도 있습니다.

출처: OpenAI, GPTs 기능 공개 (2023.11) / GPT Store 오픈 (2024.01)

일반 ChatGPT

만능 직원

뭐든 할 수 있지만, 매번 업무 배경과 규칙을 처음부터 설명해야 합니다. 반복 작업에서 효율이 떨어집니다.

GPTs

전문 담당자

자기 업무를 이미 파악하고 있는 팀원입니다. 간단한 지시만으로 원하는 결과를 즉시 만들어냅니다.

조합 전략맞춤 설정으로 기본 바탕을 깔고, GPTs로 업무별 자동화를 구축

Practice 2 — 15분

실습: 나에게 맞는 GPTs 만들어보기

'대화로 만들기' 방식을 사용합니다. GPT 빌더에게 원하는 역할을 설명하면 자동으로 GPTs가 구성됩니다.

1

GPT 만들기 화면 접속

ChatGPT 왼쪽 사이드바 → "GPT 탐색" → 상단 "+ 만들기(Create)" 클릭 → "만들기" 탭 선택

2

원하는 GPTs를 설명

아래 예시 중 하나를 선택하거나, 직접 작성하세요.

예시 A — SNS 홍보 문구 작성기
모든 답변은 한글로 해줘.
나는 작은 카페를 운영하고 있어.
새로운 메뉴나 이벤트를 인스타그램에 올릴 때 쓸 수 있는
짧고 매력적인 홍보 문구를 만들어주는 GPT를 만들고 싶어.
말투는 친근하고 감성적으로 해줘.
예시 B — 회의록 정리 도우미
모든 답변은 한글로 해줘.
나는 회사에서 회의가 끝나면 회의록을 정리해야 하는데,
내가 메모한 내용을 넣으면 깔끔하게 정리해주는 GPT를 만들고 싶어.
형식은 '날짜 / 참석자 / 안건 / 결정사항 / 후속조치'로 해줘.
직접 작성 템플릿
모든 답변은 한글로 해줘.
나는 [나의 상황]이야.
[어떤 일]을 도와주는 GPT를 만들고 싶어.
[원하는 스타일이나 조건]으로 해줘.
3

GPT 빌더 질문에 응답

GPT 빌더가 이름, 아이콘, 세부 규칙을 물어봅니다. 한국어로 자유롭게 답하면 됩니다. 잘 모르는 항목은 "알아서 정해줘"라고 답해도 문제없습니다.

4

미리보기에서 테스트 후 저장

오른쪽 미리보기 화면에서 실제 질문을 던져 결과를 확인합니다. 만족스러우면 "만들기" → "나만(Only me)" → 저장 완료합니다.
